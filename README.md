# Hai Rho — Character Reference Sheet

A single-page character reference sheet for **Hai Rho**, an Asian Palm Civet OC.

## Features

- Full body turnaround (4 plates) and face studies (6 plates)
- Flat color palette with fur, skin, and eye gradient swatches
- Glyph/tattoo library with placement notes
- Floating **Tweaks** panel (bottom-right) to switch paper tone, ink, and accent color
- Responsive layout down to mobile
